# PYUtilities-PYU
A Utility module for python that makes programming easier and has built in tools for math, ai, and others being added everyday.

⚠️ WARNING ALPHA VERSION 
Use this at your own risk. The functions might break in your programs depending on how you use them. Please leave bugs in the Issues and give any feedback or anything you would like added there too. I am open to a lot since it is my first project.


❗ HOW TO USE

To use this module download it from the releases and put it in the same folder as your program.

Parts of this project were assisted by AI tools as I continue learning and improving my development skills.
